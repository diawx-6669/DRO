# DRO Frontend

Диспетчерская панель (SPA) на React.

## Модули

- `components/DashboardMap` — интерактивная карта города с IoT-маркерами баков (Модуль 1)
- `components/BinCard` — карточка контейнера при клике на маркер
- `components/RoutePanel` — управление автопарком и назначение маршрутов (Модуль 3)
- `components/AnalyticsPanel` — бизнес-аналитика: экономия ГСМ, статистика сбора (Модуль 4)

## Запуск

```bash
npm install
npm run dev
```
